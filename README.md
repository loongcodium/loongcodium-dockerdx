# dockerdx-loong64

This project builds [Docker DX](https://github.com/docker/vscode-extension) for LoongArch with bundled binaries ([docker-language-server](https://github.com/docker/docker-language-server)).

To get the extension, search for it in the extensions tab of VSCodium and install it directly.

To install the `.vsix` file offline, please get it from [releases](https://github.com/loongcodium/dockerdx-loong64/releases), and then install it with `codium --install-extension`. Binary files are also provided.
